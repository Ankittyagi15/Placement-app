## ⚡ FASTEST WAY TO RUN (2 minutes)

### Terminal 1: Backend

```bash
cd backend
mvn spring-boot:run
```

### Terminal 2: Frontend

```bash
cd frontend
npm install
npm run dev
```

### Browser

```
http://localhost:3000
```

✅ **Done!** Application is running.
